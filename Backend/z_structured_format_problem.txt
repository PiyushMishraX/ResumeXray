The problem is occuring because google docs mentions using 
zodToJsonSchema( schemaName ) 
Which is build for zod v3 ( the latest version of zod-to-json-schema is not up to date) 
And the latest zod version is Zod v4
so instead of returning properties type and required fields it returns incomplete object


The Solution: Use Zod v4's Native z.toJSONSchema()

run - npm install zod@^4
run - npm uninstall zod-to-json-schema ( is installed )

and upgrade the gemii integration 
( remove this line if added ) - >
const { zodToJsonSchema} = require("zod-to-json-schema")

Swap -
responseJsonSchema: zodToJsonSchema( schemaName )
to
responseJsonSchema: z.toJSONSchema( schemaName )

Which will fix  it 

Reference -> https://www.buildwithmatija.com/blog/zod-v4-gemini-fix-structured-output-z-tojsonschema
